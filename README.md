# Firebase Firestore Auth

A React project for practicing authentication and Firestore-backed flows with Firebase.

## What This Project Shows
- Firebase auth integration in a client application
- Firestore-backed frontend experimentation
- Basic auth flow implementation in React

## Why This Stack / Tooling
- React provides the UI layer for auth-related screens.
- Firebase was chosen to experiment with authentication and hosted data services without building a custom backend.

## Stack
- React
- Firebase
- Firestore

## Current Status
This is a smaller practice project, but it still provides useful evidence of managed-auth and frontend integration experience.
